# **927 Technology - Bash Libraries**


### date.year

Accepts no arguments and returns a 4 digit integer for the year of the month.


&nbsp;
#### Arguments
|Short|Long|Required|Data Type|Description
|:-|:-|:-|:-|:-
NONE

&nbsp;
#### Source Command
> . ${_lib_root}/date/year.f

or

> source ${_lib_root}/date/year.f

&nbsp;
#### Example

> date.year

Returns 
> 2025
