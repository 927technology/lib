# **927 Technology - Bash Libraries**


### date.day

Accepts no arguments and returns a 2 digit integer for the day of the month.


&nbsp;
#### Arguments
|Short|Long|Required|Data Type|Description
|:-|:-|:-|:-|:-
NONE

&nbsp;
#### Source Command
> . ${_lib_root}/date/day.f

or

> source ${_lib_root}/date/day.f

&nbsp;
#### Example

> date.day

Returns 
> 19
